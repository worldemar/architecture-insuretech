# Спорные и проблемные вопросы в заданиях

Ниже описаны мои решения о спорных вопросах в задании.

- В разделе "Как подготовиться к работе" упоминается создание 6 директорий (Task1-Task6), однако в самом тексте и в разделе "Как сдать работу" описаны только 5 заданий.
    - это опечатка, заданий пять, я не буду делать шестое несуществующее задание.

- Инструментарий для схем: в задании требуется использование draw.io с иконками Yandex Cloud. 
    - Все схемы будут в PlantUML со стандартными обозначениями C4. Иконки Yandex не обязательны.

- Скриншоты для Задания 2
    - Придётся сделать вручную (и ревьюеру верить, что я их не в редакторе сделал). В зависимости от реализации можно попробовать автоматизировать процесс захвата картинок.

- География: Упоминается развитие в "регионах" РФ.
    - Задание по существу про геораспределение, достаточно предусмотреть 2-3 узла: (запад-восток) или (запад-сибирь-восток).

- Текущая инфраструктура и инструментарий: задания подразумевают установку на хост инструментов.
    - Вместо этого всё будет сделано в контейнерах docker и minikube, я не намерен на хост ставить что-то кроме докера.

- Образ из задания 2 не скачивается из-за блокировок
    - Проще всего скачать с помощью VPN + прокси
        ```
        minikube ssh "docker pull ghcr.io/yandex-practicum/scaletestapp@sha256:eff20ae3ae2d596375f9ed6d612a78d149a35a66cd2907ea90d7175ca918c993"
        ```
